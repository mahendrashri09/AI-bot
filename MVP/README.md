# AI-bot
AI Incident Copilot 

# Requirement 
litellm
chromadb
streamlit

* Description
* AI Incident Copilot Bot for SRE/DevOps, is an intelligent assistant, designed to accelerate incident resolution. It analyzes logs, retrieves similar past incidents from historical sources (in-memory DB, Jira RCA tickets, Confluence docs), and suggests actionable fixes. This reduces downtime, avoids repetitive troubleshooting, and empowers SRE/DevOps teams to resolve issues faster with AI support.
